Passwords

When Doom 64 was released gaming consoles did not have internal memory dedicated for storage. Memory cards and packs were also a new concept as well. Thus to accomodate players who did not have a memory pack, Doom 64 implemented a password system. These days it isn't as useful, but many engines still support passwords and you can get yourself out of bad or corrupted saves with them!

Password Generator
